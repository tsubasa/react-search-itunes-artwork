iTunes アルバムアートワーク検索ツール
---

React + TypeScriptの勉強資料作成がてらiTunes Storeから手軽に高画質なアルバムアートワークを保存するツールを作りました。

##  機能

- アルバムアートワークを検索
- サイズや画質を選択してアートワークを表示

## 使い方

```bash
$ yarn install && yarn start
```
